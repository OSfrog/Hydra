# HydraNote

HydraNote is a powerful, versatile, and simple note-taking application built for developers and non-developers alike. It allows you to create, edit, and manage your code snippets with markdown supported notes.

HydraNote is designed to be fast, reliable, and easy to use.

- Create, edit, and manage notes
- Support for Markdown formatting
- Lightweight and efficient

## Getting Started

### Prerequisites

Before running HydraNote, ensure you have the following software installed on your machine:

- [Node.js](https://nodejs.org/en/) (version 14 or higher)

### Installation

You have two options to run HydraNote:

1. **Run without installation** (recommended for trying it out):
  ```
  npx hydranote serve
  ```

Open your browser and navigate to `http://localhost:4005` to start using HydraNote.

2. **Install globally and run** (recommended for regular use):
  ```
  npm install -g hydranote
  hydranote serve
  ```
Open your browser and navigate to `http://localhost:4005` to start using HydraNote.

## Usage

Once HydraNote is up and running, you can create and manage your notes easily:

- Use the editor to write your note. HydraNote supports Markdown formatting.
- Your notes are automatically saved as you type.
